### CLI 명령어

* pwd : 현재 디렉토리 확인하기 (print working directory)
* cd : 디렉토리 이동하기 (change directory)
``` js
cd ..(상위경로)
cd dev(폴더명)
```
* mkdir : 디렉토리 생성하기 (make directory)
``` js
mkdir FUS13(폴더명)
```
* ls : 목록 출력하기 (List Segment)
* cat : 파일 내용 확인하기
* touch : 빈 파일을 생성
* echo : 간단한 내용이 들어있는 파일을 생성
``` js
touch index.html (폴더안에 index.html 파일 생성)
```
